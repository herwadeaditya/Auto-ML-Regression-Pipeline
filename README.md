# 🚀 Automated Machine Learning Regression Pipeline

This project builds an automated regression pipeline that:

✔ Cleans datasets  
✔ Handles missing values  
✔ Encodes categorical variables  
✔ Performs feature selection  
✔ Trains multiple regression models  
✔ Compares model performance  
✔ Selects the best model automatically  

---

## 📊 Models Used

- Linear Regression
- Ridge Regression
- Lasso Regression
- ElasticNet Regression

---

## ⚙️ Installation

pip install -r requirements.txt

---

## ▶️ Run

python auto_regression_pipeline.py
